Camera Commands:
    cam: <command>
        on --> turns camera on
        off --> turns camera off
        screenshot --> takes screenshot of image and stores it.


Sensors:

    read
        begins reading the sensor values at the default interval values. 
        if any of them are not ready then we can check that the final arrival date has been completed.
        
        time:x (time parameter specifies the number of seconds it needs to read for.)

        gp:x (gp specifies the geophone period)
        bp:x (bp species CME sensor )
        wp: (wp specifies wp)
        gl: x (how many seconds the geophone reads for)
        onlygeo:true/false

    status
        command returns information about the current sensor setup.
